# Shakti Boutique - Ladies Tailor Website

A professional website for Shakti Boutique, a ladies tailoring boutique specializing in custom Indian suits, blouses, sarees, and ready-made kurtis.

## Features

- 🏠 **Home Page** - Beautiful landing page with boutique information
- 🧵 **Custom Tailoring Services** - Showcase of all tailoring services with pricing
- 👗 **Ready-Made Products** - Collection of ready-to-wear kurtis with prices
- 📋 **Booking with Measurements** - Customers can book and provide detailed measurements (Gala, Chest, Waist, Hip, Sleeve, Length, etc.)
- 📞 **Contact Page** - Contact form and business information
- 🔐 **Admin Panel** - Secure login panel to manage services and products
- 📱 **Responsive Design** - Works perfectly on all devices
- 💾 **Local Storage** - All data is stored locally (no database needed)
